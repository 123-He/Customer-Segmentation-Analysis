# 基于RFM模型与K-Means算法的用户价值分层项目

## 项目背景与目标
**核心问题**：解决用户画像模糊、营销资源分配不均的问题。

## 核心行动
1. 数据处理：使用 Pandas 清洗并整合十万级订单数据；
2. 指标构建：搭建 RFM (Recency-Frequency-Monetary) 指标体系；
3. 聚类建模：利用 Elbow Method (手肘法) 确定最佳聚类数 K=4，完成用户自动化聚类。

## 项目成果
1. 分层结果：成功将用户划分为“重要保持”、“一般发展”等 4 类价值客群；
2. 可视化输出：通过 Seaborn 可视化不同客群的消费特征（如消费频次、客单价、最近消费时间等）；
3. 业务价值：为企业精细化运营、营销资源精准分配提供了可落地的数据分析支持。
